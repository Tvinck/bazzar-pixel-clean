# Обновление логотипов платежных систем

## Что изменилось?

Заменены текстовые плейсхолдеры на **профессиональные SVG логотипы** платежных систем в компоненте оплаты.

### Было:
- Простые цветные блоки с текстом
- Базовые стили без брендинга

### Стало:
- ✅ **T-Bank** - официальный желтый логотип
- ✅ **Visa** - синий логотип с характерным дизайном
- ✅ **Mastercard** - красно-оранжевые пересекающиеся круги
- ✅ **MIR** - зеленый логотип российской платежной системы
- ✅ **СБП** - фиолетовый градиент Системы Быстрых Платежей

## Файлы

### Новые файлы:
- `src/components/PaymentLogos.jsx` - SVG компоненты логотипов

### Измененные файлы:
- `src/components/PaymentDrawer.jsx` - интеграция логотипов

## Преимущества SVG:

1. **Векторная графика** - идеальное качество на любом разрешении
2. **Малый размер** - нет внешних изображений, все встроено в код
3. **Темная тема** - автоматическая адаптация через CSS
4. **Производительность** - мгновенная загрузка, нет HTTP запросов

## Использование

```jsx
import { TBankLogo, VisaLogo, MastercardLogo, MIRLogo, SBPLogo } from './PaymentLogos';

// В компоненте:
<TBankLogo className="w-12 h-6" />
<VisaLogo className="w-12 h-6" />
<MastercardLogo className="w-12 h-6" />
<MIRLogo className="w-12 h-6" />
<SBPLogo className="w-12 h-6" />
```

## Кастомизация

Логотипы легко кастомизировать через props:

```jsx
// Изменить размер
<VisaLogo className="w-20 h-10" />

// Добавить анимацию
<MastercardLogo className="w-12 h-6 hover:scale-110 transition-transform" />
```
